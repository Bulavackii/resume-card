# Резюме: Булавацкая Анна Алексеевна

Цифровая визитка-резюме государственного гражданского служащего.

## О проекте
Данный проект представляет собой адаптивную веб-страницу с информацией о профессиональном опыте, достижениях и навыках. Реализована поддержка версии для слабовидящих пользователей.

## Технологический стек
* **Разметка:** HTML5
* **Стили:** CSS3 (Bootstrap 5.3)
* **Иконки:** FontAwesome 6.5
* **Развертывание:** GitHub Pages
